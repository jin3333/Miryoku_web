# アプリ名 
## ホームページ作成
# 概要
知人の方の居酒屋のホームページを作成しました。
まだ実用化はしていないのですがこれから実用化予定です。
# 制作背景(意図)
ホームページを作った事がなく、勉強で友人に頼んでお店のホームページを作らせていただきました。
難しいことはしていないのですが、railsでホームページに挑戦してみたかったので今回作りました。
実際にまだ運用はしていないのですが、これから実用化できる様に改良していきます。
